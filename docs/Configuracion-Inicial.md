# Configuración Inicial 

## Requerimientos Previos

- Tener instalado NodeJs (recomendado: v24.12.0)

## Creación Inicial

- Instalación de `Vanilla TypeScript` por medio de la herramienta de desarrollo frontend `vite`, ejecutando los siguientes comandos:

```bash
    npm create vite@7 (versión recomendada)
```

Una vez ejecutado el comando de creación del proyecto, se seleccionan las siguientes opciones:

- project-name: `tmdb-explorer`
- select a framework: `Vanilla`
- Select a Variant: `TypeScript`

Después de terminada la configuración inicial, se muestran algunos comandos importantes para la ejecución del proyecto inicial.

```bash
cd tmdb-explorer
npm install
npm run dev
```
